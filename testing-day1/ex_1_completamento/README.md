# Esercizio 1: Il Carrello E-commerce

## Contesto Aziendale

Il team di frontend ha già disegnato l'interfaccia, ma il backend è incompleto. Il tuo compito è implementare la logica core per il calcolo del totale di un carrello, applicando le tasse e gli sconti corretti

## Task

Apri il file `carrello.py`. Troverai delle funzioni con la dicitura `pass` (o `NotImplementedError`).
Devi scrivere il codice per farle funzionare secondo le specifiche scritte nei commenti.

Alla fine del file c'è un blocco di codice che simula un acquisto. Se hai scritto bene il codice, eseguendo `python carrello.py` nel terminale vedrai i totali corretti.
